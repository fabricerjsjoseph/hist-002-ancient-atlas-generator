## What are How-To Guides?

How-to guides are **task-oriented** directions that guide you through solving a specific problem or accomplishing a particular goal.

### Characteristics of good how-to guides:
- ✅ Focus on achieving a specific goal
- ✅ Provide practical steps
- ✅ Assume some basic knowledge
- ✅ Are adaptable to your needs
- ✅ Solve real-world problems

### What how-to guides are NOT:
- ❌ Not tutorials (those are for learning)
- ❌ Not reference material (that's technical description)
- ❌ Not explanations (those provide understanding)

## Contributing

To add a how-to guide:

2. Start with the problem/goal
3. List prerequisites if any
4. Provide clear, numbered steps
5. Show the expected outcome
6. Update this index with a link to your guide
