## What is Reference Documentation?

Reference documentation is **information-oriented** material that provides detailed technical information about how the system works.

### Characteristics of good reference:
- ‚úÖ Structured by code/function/module
- ‚úÖ Consistent in format
- ‚úÖ Factual and accurate
- ‚úÖ Up-to-date with the codebase
- ‚úÖ Serves as lookup material

### What reference documentation is NOT:
- ‚ùå Not tutorials (those are for learning)
